# 🧛‍♂️ Darcula-VS2022

IntelliJ Darcula Theme For Visual Studio 2022 Preview 3+

## Getting Started

1. Download the `Darcula.vsix` file
2. Close all instances of VS2022
3. Double click `Darcula.vsix`
4. Start VS2022
5. Go To `Tools > Theme > darcula`
